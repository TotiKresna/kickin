## Set Up Golang
- Download [Golang](https://go.dev/dl/)
- Download [Docker](https://docs.docker.com/desktop/setup/install/windows-install/)

## Test Environment

- & `docker version`

## Set Up Project

- Buka terminal ketik `go mod tidy`
- -> `docker-compose up --build`
